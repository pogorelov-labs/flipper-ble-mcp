# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 3.0,
as its code of conduct. In short: be respectful, assume good faith, and help keep this a welcoming,
harassment-free community for everyone. Full text:
<https://www.contributor-covenant.org/version/3/0/code_of_conduct/>.

## Scope & enforcement
This applies in all project spaces (issues, pull requests, discussions) and when representing the
project publicly. Report unacceptable behavior privately via the repository's **Security → Report a
vulnerability** channel, or a direct confidential message to the maintainer. Reports are handled
discreetly and in good faith.

## Responsible-use note
This is a **dual-use** security tool. Beyond interpersonal conduct, participation requires using the
project lawfully — only against devices and targets you own or are explicitly authorized to test.
See [SECURITY.md](SECURITY.md).
